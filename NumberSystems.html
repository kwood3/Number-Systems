Breif explanation of number systems:

Binary:

Binary only uses 2 digits (base-2): 1 and 0. A binary bit is the smallest unit used by computers. In the early days of computing, the only way to enter data into a computer was by flicking switches or by feeding in punched cards or punched paper tape. A hole to represent a 1 and no hole to represent a 0.

Hexidecimal

Hexadecimal is a base-16 number system, using symbols 0-9 and A-F. The hexadecimal system is commonly used for colors (#RRGGBB (red green blue)).

Unicode

(Universal encoding) is an international encoding standard for use with different languages and scripts, by which each letter, digit, or symbol is assigned a unique numeric value that applies across different platforms and programs.

Octal

Octal is base-8, meaning it only uses 8 digits (0 through 7). In computer programming, the octal equivalent of a binary number is sometimes used to represent it because it is shorter.

ASCII

ASCII (American Standard Code for Information Interchange) is the most common format for text files in computers and on the Internet. In an ASCII file, each alphabetic, numeric, or special character is represented with a 7-bit binary number (a string of seven 0s or 1s). 128 possible characters are defined. ASCII was developed by the American National Standards Institute.

How Network Adapters Use Number Systems:

The network adapter uses computer drivers to setup the wireless network adapter. When the wireless network adapter is plugged into the pci port. Its then identified by two IDs which consist of 4 hexadecimal numbers each. The vendor id helps the network adapter communicate with the software inside the system.
